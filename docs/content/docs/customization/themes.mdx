# Themes

Vortex is entirely customizable! You can customize the look and feel of your blog by the values listed below.

<Spacer y={2} />

## Usage

```json
{
  "textColor": "black",
  "twitter": {
    "theme": "dark"
  },
  "codeBlockTheme": "atomOneDark",
  "primary": "#23272a",
  "dateColor": "white",
  "inlineTextColor": "#333",
  "bgColor": "white",
  "linkColor": "#5862f2",
  "inlineBgColor": "#ebebeb"
}
```

<Spacer y={2} />

### Theme Props

| Attribute          | Nullable | Accepted values                                                           | Default       |
| ------------------ | -------- | ------------------------------------------------------------------------- | ------------- |
| **textColor**      | true     | `color`                                                                   | `#000`        |
| **twitter-theme**  | true     | `dark/light`                                                              | `dark`        |
| **codeBlockTheme** | true     | [Supported Themes](/docs/customization/themes#supported-codeblock-themes) | `atomOneDark` |
| **primary**        | true     | `color`                                                                   | `#23272a`     |
| **dateColor**      | true     | `color`                                                                   | `#fff`        |
| **bgColor**        | true     | `true/false`                                                              | `white`       |
| **linkColor**      | true     | `color`                                                                   | `#5862f2`     |
| **inlineBgColor**  | true     | `color (in hex)`                                                          | `#ebebeb`     |
