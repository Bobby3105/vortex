# Getting started

Welcome to the Vortex documentation!

Vortex allows you to make beautiful, customizable and modern blogs, make sure to checkout our [Getting Started](/docs/layouts/default) to get all the supported layouts available!

<Spacer y={2} />

### Installation

Inside your React project directory, install Vortex by running either of the following:

```bash
yarn add @hydralite/vortex
# or
npm i @hydralite/vortex
# or
pnpm i @hydralite/vortex
```
